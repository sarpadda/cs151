1) Navigate to https://github.com/LGoodDatePicker/LGoodDatePicker/releases 
2) Scroll and download 'LGoodDatePicker-10.4.1.jar'
3) Once all src files have been imported to Java Project, right click on the Java Project - > Build Path - > Add External Archives
4) Locate the jar file downloaded in step 2. 
5) After this done, you should be able to see a Referenced Library directory in your Eclipse workspace. 

Optional: 
Create a folder inside of the Java Project and name it 'image_data'. Drag and drop the image in the folder named 'image_data' into that folder in your java project.
